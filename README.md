# RevealJS + React + TailwindCSS
A heavily customizable presentation slides template using [Revealjs](https://revealjs.com/) and featuring [React](https://react.dev) and [Tailwind CSS](https://tailwindcss.com).
## Features
- [x] Nicely styled headings, lists, transitions and a __rich ecosystem__ of resources thanks to [Revealjs](https://revealjs.com/).
- [x] Full customization power of __React__ and __Tailwind CSS__
- [x] Easily __build re-useable slide components__ or __templates__!
- [x] $\LaTeX$ support thanks to [https://katex.org/](https://katex.org/)
- [x] Add insighful and fun __interactivity__ to your Presentation, based on all the React knowledge you (might) already have
- [x] Effortlessly _animate the drawing of SVG images_ on your slides using [https://github.com/maxwellito/vivus](https://github.com/maxwellito/vivus)
- [x] __Annotate slides__ while presenting, powered by [https://excalidraw.com/](https://excalidraw.com/)
- [x] Full control: No limitations of what can and cannot be included your slides. Create fully custom slides at will.
